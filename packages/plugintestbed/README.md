## Quick start

```sh
npm install --save-dev @knodes/typedoc-plugintestbed typedoc@^0.23.0
```

## Compatibility

This plugin version should match TypeDoc `^0.23.0` for compatibility.

> **Note**: this plugin version was released by testing against `^0.23.28`.
